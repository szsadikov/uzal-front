# Мок-данные (MSW)

Все API-запросы в dev-режиме перехватываются Mock Service Worker.

## Структура

- `src/mock/fakeApi/` — хендлеры эндпоинтов
- `src/mock/data/` — статические тестовые данные (`.ts` / `.json`)
- `src/mock/index.ts` — общий индексный файл, подключает все хендлеры

## Добавление нового эндпоинта

1. Создай или обнови хендлер в `src/mock/fakeApi/`
2. Добавь тестовые данные в `src/mock/data/`
3. Подключи хендлер в `src/mock/index.ts`

## Правила

- Мок-данные должны отражать реальную структуру API — те же поля и типы
- **Не трогать** `public/mockServiceWorker.js` — генерируется автоматически командой MSW
- Для тестирования конкретной роли: поменяй `role` в `src/mock/fakeApi/authFakeApi.ts` и перелогинься
